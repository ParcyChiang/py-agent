## 写作前准备

1. **备份规则**：每次写论文内容前，先将原文档复制一份作为备份（只读文件，永远保留上一版本）
2. **读取批注**：阅读文档中的所有批注，了解格式要求和修改意见
3. **格式优先**：严格按照文档指定的格式写入内容

## 操作流程

1. 复制原文档为备份（只读）
2. 读取原文档的批注内容
3. 按批注要求格式写入新内容

## 工具与方法

### 1. 备份原文档

**命令**：
```bash
cp "原始文档路径" "备份目录/原文件名_backup_$(date +%Y%m%d_%H%M%S).扩展名"
chmod 444 "备份目录/原文件名_backup_"*.扩展名
```

**示例**：
```bash
cp "/Users/rubyjiang/Documents/学习/毕设/.../论文.dotx" "/Users/rubyjiang/Documents/学习/毕设/.../论文_backup_$(date +%Y%m%d_%H%M%S).dotx"
```

### 2. 读取 .docx/.dotx 文档内容

**方法**：使用 python-docx 库读取 .docx 文件

```bash
source .venv/bin/activate
python3 << 'EOF'
from docx import Document
doc = Document('文档路径.docx')
for i, para in enumerate(doc.paragraphs):
    if para.text.strip():
        print(f'[{i}] {para.text}')
EOF
```

**注意**：.docx 本质是 zip 文件，可用以下方法直接读取 XML：
```bash
python3 << 'EOF'
import zipfile
import xml.etree.ElementTree as ET

with zipfile.ZipFile('文档.dotx', 'r') as z:
    content = z.read('word/document.xml').decode('utf-8')
    # 解析XML提取文本...
EOF
```

### 3. 读取批注

**方法**：python-docx 的 `doc.comments` 属性（需先安装库）

**注意**：某些 .dotx 模板没有批注

### 4. 写入文档

**方法**：使用 python-docx 创建新文档并设置格式

```python
from docx import Document
from docx.shared import Pt, Cm
from docx.enum.text import WD_ALIGN_PARAGRAPH
from docx.oxml.ns import qn

doc = Document()

# 标题
title = doc.add_paragraph()
title_run = title.add_run('标题文字')
title_run.font.size = Pt(22)  # 小2号
title_run.font.name = '黑体'
title_run._element.rPr.rFonts.set(qn('w:eastAsia'), '黑体')
title.alignment = WD_ALIGN_PARAGRAPH.CENTER

# 摘要内容
abstract_content = doc.add_paragraph()
abstract_run = abstract_content.add_run('摘要正文...')
abstract_run.font.size = Pt(10.5)  # 5号
abstract_run.font.name = '宋体'
abstract_run._element.rPr.rFonts.set(qn('w:eastAsia'), '宋体')
abstract_content.line_spacing = 1.5

# 分页
doc.add_page_break()

# 保存
doc.save('输出路径.docx')
```

### 5. 字体格式对应

| 模板要求 | 实际字号 |
|---------|---------|
| 1号 | 约26pt |
| 小2号 | 22pt |
| 2号 | 22pt |
| 3号 | 16pt |
| 4号 | 14pt |
| 5号 | 10.5pt |
| 小5号 | 9pt |

### 6. 常见问题

- **.doc 格式**：老式二进制格式，python-docx 不支持，需转换或用其他工具
- **.dotx 模板**：本质是 .docx，但 content-type 为 template，需复制为 .docx 后读取
- **中文字体**：必须同时设置 `font.name` 和 `rPr.rFonts.set(qn('w:eastAsia'), '字体名')`
